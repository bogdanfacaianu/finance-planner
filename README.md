# Financial Planner App

React + Supabase (RLS/Auth)

## Features
- Expense and budget tracking
- Shared car-ride expenses
- Secure authentication

## Setup Instructions
1. Clone repo
2. `npm install`
3. Set Supabase credentials in `.env`
4. Run: `npm run dev`
5. Deploy: `npm run deploy`

## Technologies
- React (Vite)
- Supabase (Postgres/Auth/RLS)
- GitHub Pages
